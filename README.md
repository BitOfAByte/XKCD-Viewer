# XKCD-Viewer
simple completely terminal viewer for XKCD

# Installation

Requires:
* Python 3
* Beautiful Soup 4
* drawille (http://github.com/asciimoo/drawille)[asciimoo/drawille]
* requests
* PIL
* Curses

# Running

`python XKCDViewer.py`
