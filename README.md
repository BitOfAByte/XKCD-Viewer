# XKCD-Viewer
simple completely terminal viewer for XKCD

# Installation

Requires:
* Python 3
* Beautiful Soup 4
* drawille [asciimoo/drawille](http://github.com/asciimoo/drawille)
* requests
* PIL
* Curses

# Running

`python XKCDViewer.py`
some comics will probably break this, but only standard ones have been tested
